# About Guardrail

Guardrail is a security and compliance platform for AI systems. The Guardrail API delivers
real-time policy enforcement for LLM inputs and outputs while producing audit trails and evidence
packages suitable for regulated environments.

**Status:** Patent Pending (U.S.)
**Primary components:** Core Runtime, Enterprise Admin Console, Verifier Service, Policy Packs.
